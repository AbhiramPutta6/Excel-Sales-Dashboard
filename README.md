# 📊 Excel Interactive Sales Dashboard

## Overview
An interactive sales dashboard built in Excel analyzing store sales performance 
by gender, age group, sales channel, and order status.

## Dashboard Preview
Excel-Sales-Dashboard
/excel sales dashboard.png

## Tools Used
- Microsoft Excel
- Pivot Tables
- Charts (Bar, Pie, Line)
- Slicers for interactivity

## Key Features
- Interactive filters (Month, Category, Channel)
- Sales breakdown by gender and age group
- Top performing sales channels
- Order status analysis
- Monthly sales trend

## Key Insights
- Women customers contributed ~65% of total sales, significantly outperforming men
- Adults (30-49 years) are the biggest buyers, contributing ~50% of total orders
- Amazon, Flipkart, and Myntra together contributed ~80% of total sales
- Maharashtra, Karnataka, and Uttar Pradesh are the top 3 states by orders

## How to Use
1. Download the `.xlsx` file
2. Open in Microsoft Excel
3. Use the slicers to filter data interactively
